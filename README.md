## Tutorial on how to use the gather_image.py (windows)
- open command prompt
- python gather_images.py <label> <amount> \n
  example: \n
  <b>python gather_images.py rock 250<b> \n
  it will open your webcam \n
  press a to gather images \n
  press q to quit
